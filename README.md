## **Описание**
YouTube Parser Ultimate — это удобный десктопный парсер для Windows и Linux, который позволяет собирать полную информацию о каналах и плейлистах YouTube.  
Программа получает реальные просмотры и лайки видео, предоставляет визуализацию, сортировку, фильтрацию и возможность экспорта данных в Excel или CSV.

---

## **Основные возможности**
- Парсинг канала или плейлиста YouTube с получением:
  - Названия видео
  - Ссылок на видео
  - Реальных просмотров
  - Реальных лайков
- Сбор информации о канале:
  - Название канала
  - Количество подписчиков
  - Количество видео
  - Аватар канала
- Параллельный парсинг с настройкой количества потоков (workers)
- Сортировка видео по просмотрам или лайкам
- Фильтрация по минимальному количеству просмотров
- Экспорт данных в Excel или CSV
- Просмотр и копирование ссылок на видео
- Загрузка видео с выбором качества (best/audio/worst)
- Анимация графиков просмотров, лайков или вовлеченности
- Поддержка вставки ссылки из буфера обмена
- Темная и светлая тема интерфейса

## **Установка**

### 1. Установите Python
Скачайте Python 3.10 или выше с [официального сайта](https://www.python.org/downloads/).  
Обязательно отметьте галочку **“Add Python to PATH”** при установке.

### 2. Установите зависимости
Откройте терминал или PowerShell и выполните команду:

```bash
pip install customtkinter yt-dlp requests pillow pandas matplotlib pyperclip
```

### 3. Скачайте код
Клонируйте репозиторий и перейдите в папку проекта:

```bash
git clone <ваш_репозиторий>
cd <папка_репозитория>
```

### 4. Запуск парсера

```bash
python main.py
```
